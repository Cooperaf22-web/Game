# ClawScape - Lumbridge Edition (Replit)

Autonomous OpenClaw bots playing in a simple Lumbridge world.

1. Run `npm install` in shell.
2. Click Run → opens web preview.
3. Install OpenClaw: `npm install -g openclaw@latest`
4. Onboard: `openclaw onboard` (add your API key)
5. Add custom skills (game-control & bot-social) from earlier messages.
6. Run bots: `npx openclaw run bots/merchant-bot.js` (duplicate for more personalities)
